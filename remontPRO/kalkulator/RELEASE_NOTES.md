# Release notes

## v1.1.7 — Posadzki / przygotowanie podłoża
- Dodano niezależny moduł `Posadzki / przygotowanie podłoża`.
- Tryb szybki: m² z obmiaru, gruntowanie, wylewka samopoziomująca, wyrównanie lokalne, szlifowanie, naprawa pęknięć.
- Tryb szczegółowy: tabela per pomieszczenie z pobieraniem m² z obmiaru i opcją korekty ręcznej.
- Dodano stawki administracyjne dla gruntowania podłoża, samopoziomu cienkiego/grubszego, wyrównania lokalnego, szlifowania i naprawy pęknięć.
- Integracja z podsumowaniem, standardem wykonania, VAT i buforem.
- Tooltipy dla kluczowych pól modułu.

## v1.1.6 — Podłogi + listwy
- Dodano niezależny moduł `Podłogi + listwy przypodłogowe`.
- Tryb szybki: powierzchnia podłóg, listwy, progi, typ podłogi, podkład, folia, demontaż listew, docinki.
- Tryb szczegółowy: tabela per pomieszczenie z pobieraniem m² i mb z obmiaru.
- Dodano stawki administracyjne dla paneli laminowanych, winylu click, deski warstwowej, podkładu, folii, listew, progów i demontażu listew.
- Integracja z podsumowaniem i buforem/standardem.
- Tooltipy dla kluczowych pól modułu.
